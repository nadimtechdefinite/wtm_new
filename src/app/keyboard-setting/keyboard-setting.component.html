Adding a Regional Language Keyboard in System

Step-by-step guide to add and use regional language keyboard

Step 1
Go to Settings of the System

Open the system settings from the Start menu or control panel.

Step 2
Select “Time and Language”

Click on Time and Language from the left-hand side menu.

Step 3
Select “Language and Region”

Choose Language and Region from the available options.

Step 4
Click on “Add a language”

Click the Add a language button to proceed.

Step 5
Language selection screen appears

A new screen will appear showing available languages.

Step 6
Search required language

Type the required language (for example: Hindi) in the search box and click Next.

Step 7
Click on Install

Click the Install button to add the language.

Step 8
Keyboard download

The selected language keyboard will be downloaded and installed.

Step 9
Change keyboard language

Click on ENG from the bottom toolbar of the screen.

Step 10
Select required language

Select the required language from the list and start typing.

Note: The user will now be able to type in the selected regional language and submit the grievance successfully.
